# NightPatch
Enable Night Shift on any old Mac models.

## Supported macOS version

- macOS 10.12.4 (16E195)

- macOS 10.12.5 DP1 (Coming soon)

## Requirements

- Xcode 8.3 or higher with Command Line Tools.

- Unsupported old Mac

## Usage

`$ cd /path/to/NightPatch`

`$ ./NightPatch.sh`

## How to revert

`$ ./NightPatch.sh -revert`
